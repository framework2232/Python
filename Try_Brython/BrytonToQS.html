QS File Template

QS File Template is for anyone who is building a website from complete scratch using HTML and CSS. QS File Template reduces time in the basic setup of file structure by building a common folders and adding blank basic files.\nBy default, the files and folders include - index.html, style.css, robots.txt, mainfest.json and main.js.

Click 'BUILD' for default, or deselect as required.
